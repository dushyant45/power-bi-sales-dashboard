# Power BI Sales Analysis Dashboard

## Project Overview
This project provides a comprehensive analysis of sales trends and performance. It is designed to help stakeholders identify key growth drivers and optimize sales strategies using data-driven insights.

## Tech Stack
* *Tool:* Power BI Desktop
* *Data Source:* Excel/SQL (mention your specific source)
* *Concepts:* Power Query (ETL), Data Modeling (Star Schema), DAX, Data Visualization.

## Key Features & Insights
* *Data Cleaning:* Handled missing values and transformed data using Power Query.
* *DAX Measures:* Developed measures for Total Sales, YOY Growth, and Profit Margins.
* *Interactivity:* Integrated Slicers and Drill-through features for deep-dive analysis.
